#To use Django CRUD:
Install Python if you dont do.
Instal pip if you dont do.
Install Django:
	pip install Django
Install django-tastypie
Run migration:
python manage.py makemigrations
python manage.py migrate
Run server:
python manage.py runserver

http://localhost:8000/api/movies
